Practicando con git

I am creating a blogpost as example to work with git

En esta clase aprenderemos como revertir un commit sin usar forzosamente el commando git reset

Al parecer para efectuar cambios y verlos reflejados en git deben hacerse sobre un archivo de texto, es decir creando contenido más no añadiendole estilos.
He notado que si quiero reflejar lo editado tanto en un archivo como en el otro lo debo guardar en conjunto, es decir, hacer el commit cuando ya tenga mis cambios listos

Retomando los conceptos de git restore y git reset
